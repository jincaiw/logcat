# v1.5.3 更新日志

## 发布日期
2026-04-20

## 主要更新

### 1. Syslog 推送功能优化

#### 1.1 支持选择推送字段
- 可通过推送消息模板选择需要推送的字段
- 只推送选中的字段，不再包含完整的原始日志信息
- 支持多级字段匹配，自动识别中文/英文字段名

#### 1.2 支持多种输出格式
- **JSON 格式**：输出为标准 JSON 对象
- **RFC 3164 格式**：输出为 BSD syslog 格式
- **RFC 5424 格式**：输出为新版 syslog 格式
- 未选择字段时，默认输出完整的告警信息

#### 1.3 值转换支持
- 支持字段值转换规则
- 例如：防护状态（1→未拦截，2→已拦截，3→执行中）

#### 1.4 字段映射增强
- 修复了字段映射时可能丢失未配置字段的问题
- 支持多级 fallback 匹配，确保字段正确映射
- 自动识别解析后的中文字段名与映射配置的对应关系

#### 1.5 UDP 数据包处理
- 自动处理超过 65507 字节的 UDP 数据包
- 对大字段进行自动截断（超过 500 字符截断）
- 截断后仍超限时保留核心字段子集

### 2. 筛选策略白名单功能

#### 2.1 白名单配置
- 支持在筛选策略中配置 IP 白名单
- 支持 CIDR 格式（如 `192.168.1.0/24`）
- 支持配置单个 IP 地址
- 可指定白名单匹配的字段（默认：源 IP）

#### 2.2 白名单应用
- 匹配白名单的日志直接跳过，不推送告警
- 适用于内网可信 IP 的过滤场景

## 其他优化

- 前端类型定义优化，统一使用 models.ts 中的类型
- 增强调试日志，方便问题排查

## 修复的问题

- 修复了字段映射时部分字段丢失的问题
- 修复了 UDP 包超限时整条消息发送失败的问题
- 修复了值转换规则未生效的问题

## 编译说明

### Windows 桌面版
```bash
wails build -platform windows/amd64
```

### macOS 桌面版
```bash
# ARM 版本（M1/M2/M3）
wails build -platform darwin/arm64

# 通用版本（ARM + Intel）
wails build -platform darwin/universal
```

### Linux Web 版
```bash
bash build-web.sh
```

## 升级建议

1. **备份数据库**：升级前建议备份 `syslog2bot.db`
2. **数据库迁移**：新版本会自动创建白名单字段，无需手动迁移
3. **配置检查**：升级后建议检查推送模板的字段选择是否正确
