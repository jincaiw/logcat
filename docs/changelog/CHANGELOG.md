# 更新日志

## v1.5.0 (2026-03-25)

### 核心功能升级

#### Debug 追踪模式

- 日志 ID 全链路追踪功能
- 可视化展示日志处理流程：接收 → 解析 → 筛选 → 推送
- 方便快速定位告警未推送原因

#### 数据统计模块

- 按筛选策略统计字段值分布
- Top N 分布图可视化展示
- 支持导出 CSV 和复制 IP 列表

### 跨平台支持

- 支持 macOS 桌面版（Intel/Apple Silicon）
- 支持 Windows 桌面版
- 新增 Linux Web 版本，支持浏览器访问

### 消息推送

#### 多 IM 消息平台支持

- 钉钉机器人推送（Webhook + 加签验证）
- 飞书机器人推送（Webhook + 加签验证）
- 企业微信机器人推送（Webhook + Key）
- 邮件推送（SMTP）

#### Syslog 日志推送

- 支持 UDP/TCP 协议
- 支持多种输出格式：JSON、RFC 3164、RFC 5424
- 自动添加转发标记，防止循环转发

### 操作流程优化

#### 推送配置优化

- 将「机器人配置」和「告警推送」模块合并为「数据推送」
- 简化配置流程，减少模块切换
- 推送测试错误提示优化

#### 筛选策略优化

- 字段选择器可直接从关联的解析模板中选择字段

#### 告警规则优化

- 一个推送配置支持添加多个告警规则
- 每个规则可关联不同的筛选策略和消息模板
- Syslog 推送支持选择输出格式

### 数据库并发优化

- WAL 模式提升数据库性能
- 解决高并发下的写入冲突问题
- 批量写入优化日志存储性能
- 日志列表查询优化，添加索引

### UI/UX 优化

- 下拉框、输入框样式统一
- 卡片布局优化
- 表格深色模式样式统一

### Bug 修复

#### Web 版本

- 修复端口占用问题（Ctrl+C 后端口未释放）
- 修复数据统计模块无数据问题
- 修复系统状态卡片无数据问题
- 修复筛选策略解析模板显示 no data 问题

#### 推送配置

- 修复 Syslog 平台显示为"钉钉"问题
- 修复 Syslog 推送测试按钮无法工作问题

#### 其他

- 修复 Mac 版本 IP 地址显示虚拟网络接口问题
- 修复深色模式下部分样式显示异常
- 修复告警推送偶发失败问题
